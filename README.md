# Screen Lookup

A modern, multilingual movie, TV show, and people information lookup application.

## Features

- **Universal Search**: Search movies, TV shows, and people in one place
  - Real-time search suggestions as you type
  - Displays poster/profile, title/name, year/department, and rating
  - Intelligent filtering with media type badges
  - Instant results for all content types
- **Detailed Information**: View comprehensive details including cast, crew, trailers, and more
- **Multi-language Support**: Available in 11 languages (English, Chinese, Japanese, Korean, Spanish, French, German, Russian, Italian, Portuguese)
- **External Links**: Quick access to IMDb, TMDB, Trakt, Letterboxd, Metacritic, and more
- **Streaming Availability**: See where to watch movies and TV shows

## Usage

1. Enter a search term (movie, TV show, or person name)
2. Browse unified search results with media type indicators
3. Select from results or press Enter to view the first result
4. Explore detailed information, cast, crew, and external links
